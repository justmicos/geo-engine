# GEOEngine Knowledge Base

## GEO Content Engineering

GEOEngine orchestrates knowledge bases, AI generation, review/publishing, multi-site distribution, and analytics into a sustainable content operations pipeline.

## Core Pipeline
1. Upload trusted materials, auto-chunk, vectorize via pgvector
2. Title libraries, keyword libraries, images, authors management
3. AI content generation with RAG context from knowledge base
4. Draft, review, publish, recycle workflow
5. Multi-site distribution via Agent protocol
6. Analytics for views, crawler detection, distribution status

## Technical Stack
- PHP 8.2+ / Laravel 12
- PostgreSQL with pgvector
- Redis (queue, cache)
- Docker Compose (5 services)
